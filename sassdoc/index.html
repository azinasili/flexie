    flexie-column

    @mixin flexie-column($width: auto, $gutter: map-get($flexie, gutter), $display: flex, $direction: column, $align: null, $order: null) { ... }

    Description

    Create column element

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $width

    Column width

    		Numberauto
    $gutter

    Gutter width

    		Numbermap-get($flexie, gutter)
    $display

    Flex display value

    		Stringflex
    $direction

    Flex direction value

    		Stringcolumn
    $align

    Flex alignment value

    		Stringnull
    $order

    Source order value

    		Stringnull

    Output

    Width, padding, display, flex-direction, align-self, order, and flex-shrink declarations to create column

    Example

      .foo {
    @include flexie-column(3/5, 20px, flex, row, center, 2);
  }
      .foo {
    flex-basis: 60%;
    flex-grow: 0;
    max-width: 60%;
    padding-left: 10px;
    padding-right: 10px;
    display: flex;
    flex-direction: row;
    align-self: center;
    order: 2;
  }

    Requires

    • [mixin] flexie-width — Column width setter
    • [mixin] flexie-gutter — Row or column gutter width setter
    • [mixin] flexie-display — Flex display property setter
    • [mixin] flexie-direction — Flex direction property setter
    • [mixin] flexie-align — Flex alignment setter
    • [mixin] flexie-order — Source order setter

    Used by

    • [mixin] flexie-col

    Description

    Create row element

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $display

    Flex display value

    		Stringflex
    $direction

    Flex direction value

    		Stringrow
    $wrap

    Flex wrap value

    		Stringwrap
    $gutter

    Gutter width

    		Numbermap-get($flexie, gutter)
    $justify

    Justify-content value

    		Stringnull
    $align

    Flex alignment value

    		Stringnull

    Output

    Display, flex-direction, flex-wrap, margin, justify-content, and align-items declarations to create row

    Example

    flex-direction: row-reverse;
    flex-wrap: nowrap;

    margin-right: -10px;
    justify-content: center;
    align-items: stretch;
  }
