    yoga-align

    @mixin yoga-align($row-or-col: row, $align: flex-start) { ... }

    Description

    Flex alignment setter

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $row-or-col

    Determine alignment context

    		Stringrow
    $align

    Flex alignment value

    		Stringflex-start

    Output

    Align-self or align-items declaration depending on context passed

    Example

      .foo {
    @include yoga-align(col, stretch);
  }
      .foo {
    align-self: stretch;
  }

    Throws

    • Warning and invalid property if $row-or-col is not 'r', 'row', 'c', 'col', or 'column'

    Used by

    • [mixin] yoga-column
    • [mixin] yoga-row

    yoga-column

    @mixin yoga-column($width: auto, $gutter: map-get($yoga, gutter), $display: flex, $direction: column, $align: null, $order: null) { ... }

    Description

    Create column element

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $width

    Column width

    		Numberauto
    $gutter

    Gutter width

    		Numbermap-get($yoga, gutter)
    $display

    Flex display value

    		Stringflex
    $direction

    Flex direction value

    		Stringcolumn
    $align

    Flex alignment value

    		Stringnull
    $order

    Source order value

    		Stringnull

    Output

    Width, padding, display, flex-direction, align-self, order, and flex-shrink declarations to create column

    Example

      .foo {
    @include yoga-column(3/5, 20px, flex, row, center, 2);
  }
      .foo {
    flex-basis: 60%;
    flex-grow: 0;
    max-width: 60%;
    padding-left: 10px;
    padding-right: 10px;
    display: flex;
    flex-direction: row;
    align-self: center;
    order: 2;
  }

    Requires

    • [mixin] yoga-width — Column width setter
    • [mixin] yoga-gutter — Row or column gutter width setter
    • [mixin] yoga-display — Flex display property setter
    • [mixin] yoga-direction — Flex direction property setter
    • [mixin] yoga-align — Flex alignment setter
    • [mixin] yoga-order — Source order setter

    Used by

    • [mixin] yoga-col

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $display

    $direction

    		Stringrow
    $wrap

    $gutter
