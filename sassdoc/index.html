    flexie-gutter

    @mixin flexie-gutter($row-or-col: 'col', $flexie-gutter: map-get($flexie, gutter)) { ... }

    Description

    Row or column gutter width setter

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $row-or-col

    Determine gutter width context

    		String'col'
    $flexie-gutter

    Gutter width

    		Numbermap-get($flexie, gutter)

    Output

    Gutter width for row or column depending on context passed

    Example

      @include flexie-gutter();
  @include flexie-gutter(30px);
  @include flexie-gutter(30px, col);
